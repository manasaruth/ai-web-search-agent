# AI Web Search Agent

## Overview
This project is an AI-powered web search agent that retrieves real-time information from the internet and generates answers using an LLM.

## Features
- Accepts natural language queries
- Retrieves top web results using DuckDuckGo
- Generates answers based on retrieved content
- Displays source links

## Tech Stack
- Python
- DuckDuckGo Search API
- Google GenAI (Gemini)

## How to Run

1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   python app.py

## Note
Due to API quota limitations, a fallback response is currently used. The system is fully functional with valid API access.
